cawtooth — PSID playback

Plays a Commodore 64 .sid file end-to-end through a vendored fake6502 CPU emulator driving our reSID SID chip emulator inside a single AudioWorklet. Matthew Cannon's "Batman, the Movie" ships as a built-in sample; use the file picker to drop in any other PSID v1–v4 tune.

Per-voice oscilloscope

Each canvas shows one SID voice directly from reSID, sampled at the output rate. Single-SID tunes show three canvases; v3/v4 PSIDs that use multiple SID chips show up to nine (one row per chip). Amplitude is auto-scaled per voice so a single gated note fills its canvas even though its raw level is well below the chip's mix peak.
