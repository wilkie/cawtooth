cawtooth — HERAD player

Load an .HSQ, .AGD, or .HA2 file from a Cryo Interactive game (Dune, KGB, MegaRace, etc.). The file is HSQ-decompressed, parsed into patterns and instruments, and rendered into an OPL register stream that plays through our emulator.

All standard HERAD playback features are supported: program changes, note on/off, pitch bend (fine + coarse), slide, transpose, velocity macros, aftertouch (v1), AGD panning, and v2 keymap indirection. HERAD v2 files that don't use keymaps (like ALARME) auto-detect as v1 — select v2 from the variant menu if a file sounds wrong.
