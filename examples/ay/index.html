cawtooth — AY register-dump & SNDH playback

Plays an AY-3-8910 / YM2149 register-dump file (.psg, .vtx, .ym), an ASC Sound Master tracker module (.asc), or an Atari ST .sndh binary through CawtoothPlayer's auto-detect path. Register-dump files are sniffed by magic, decompressed (LH5 for VTX/YM), and de-interleaved into a register-write event stream; ASC modules are first parsed and then run through cawtooth's in-process replay engine; SNDH files are loaded into a simulated Atari ST and played back by running their original m68k code on Musashi with the YM register writes routed straight to Ayumi. A built-in synthetic PSG sample loads on first play if nothing else is selected.

Per-voice oscilloscope

Three canvases — one per AY tone channel (A / B / C). Samples are pre-pan, pre-mix DAC values from Ayumi, sampled at the audio output rate. Amplitude is auto-scaled per voice so a single gated note fills its canvas even though its raw level sits well below the chip's mix peak.
